# i18n orphan baseline (de.json)
# Wave 2 - T-W2-I18N-ORPHAN-CI-TEST
#
# This file captures every i18n key in data/i18n/de.json that currently has
# no quoted-literal reference under src/**/*.{cs,xaml}. It is a regression
# baseline. The CI guard Wave2I18nOrphanGuardTests asserts:
#
#   * The set of orphaned keys today MUST NOT grow beyond this baseline.
#   * The baseline MAY shrink (entries must be removed when a key is either
#     wired up or deleted). A dedicated future sweep is expected to drive
#     this list to zero.
#
# Adding a new orphan to de.json without removing it through code is a
# regression and breaks the build. When that happens, either (a) reference
# the new key in src/ or (b) remove it from de.json before commit.
#
# This baseline reflects the post-T-W1-I18N-ORPHAN-SWEEP state. The Wave-1
# sweep removed only the cull-driven slice (Plugin / ScreenScraper /
# RetroAchievements / Frontend-Export); the bulk of legacy WPF tab/string
# orphans is a separate cleanup task that is not in Wave 2 scope.
