# Sponsor Romulus

Romulus is free and open-source software (GPLv3).  
If it saves you hours of manual collection work, consider supporting ongoing development.

---

## GitHub Sponsors

**Tier 1 – Supporter** · $3/month  
- Name in CONTRIBUTORS.md  
- Early access to release notes

**Tier 2 – Patron** · $8/month  
- All Tier 1 perks  
- Vote on feature priority in GitHub Discussions  
- Access to development diary (GitHub Discussions pinned posts)

**Tier 3 – Backer** · $20/month  
- All Tier 2 perks  
- Priority issue response (48h target)  
- Access to private beta releases

---

## One-time donations

GitHub Sponsors also supports one-time donations.  
Any amount is appreciated and goes directly toward hosting, tooling and development time.

---

## What the money covers

- Windows code-signing certificate (required for trusted installs)  
- Hosting for future update/DAT catalog feeds  
- Development tooling licenses  
- Community infrastructure (Discord, forum)

---

## Romulus will always be free

The core tool — GUI, CLI, API — will remain GPLv3 and free forever.  
Sponsorship is purely voluntary and funds quality-of-life improvements.
